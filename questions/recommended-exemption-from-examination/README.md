## 专业信息

### 信息科学与技术学院

#### 软件工程

软件工程保研通常在大三下半学期出结果，依据是学习成绩、奖项（较为厉害的，如国奖）和科研成果（主要是一作论文）。将本人前五学期（也可能包含第六学期，即大三下半学期）在上面三个方面的成就按照一定比率折算为分数（通常学习成绩占比最高，往往能达到 `90%` 以上），在专业范围内排名。排名占比靠前的学生获得保研资格。

软件工程保率在 `15%` 左右，约 `20` 人（全专业约 `150` 人）。
